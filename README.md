# Bonk.io-skin-scraper

# What does it do
It enters the lobbies that are currently open in bonk.io and scrapes the skins used by the players in those lobbies.

# How to use it
Install the packages in requirements.txt and then run main.py

Sometimes you will get ratelimited, that is expected and will be logged.

# How to use the skins

Download the popular tampermonkey extension "Bonk Commands"

In the chat of a lobby, type "/eval overideSkin skin" here, replace skin with the json at the "avatar" part of each player. Just copy paste that avatar json and rejoin the lobby
